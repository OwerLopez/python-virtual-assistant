# Python Virtual Assistant

![Python](https://img.shields.io/badge/Python-3-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Speech](https://img.shields.io/badge/Speech-Recognition-FF6F00?style=for-the-badge)

Voice-enabled virtual assistant built in **Python**: listens to voice commands, answers with text-to-speech and automates everyday tasks.

## Capabilities

- 🎙️ Speech-to-text command recognition (Spanish)
- 🔊 Text-to-speech responses
- ⚙️ Task automation: web search, apps, system queries

## Quick start

```bash
pip install -r requirements.txt  # speechrecognition, pyttsx3, pyaudio
python main.py
```
